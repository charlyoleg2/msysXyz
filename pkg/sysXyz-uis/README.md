sysXyz-uis
==========


Presentation
------------

*sysXyz-uis* is the mini-server package for enabling the local installation of the web-ui *sysXyz-ui*.


Requirements
------------

- [node](https://nodejs.org) > 22.0.0
- [npm](https://docs.npmjs.com/cli) > 11.0.0


Installation
------------

```bash
npm i -D sysXyz-uis
```


Usage
-----

```bash
npx sysXyz-uis
npx sysXyz-uis --help
```


Usage without installation
--------------------------

```bash
npx sysXyz-uis
npx --package=sysXyz-uis sysXyz-uis
npx --package=sysXyz-uis sysXyz-uis --help
```
