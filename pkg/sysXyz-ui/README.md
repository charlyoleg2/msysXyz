sysXyz-ui
=========


Presentation
============

*sysXyz-ui* is the web-ui of *sysXyz*.


Dev
===

*sysXyz-ui* is built with [svelte](https://svelte.dev/).

```bash
git clone https://github.com/charlyoleg2/msysXyz
cd msysXyz
npm i
npm -w sysXyz-ui run clean
npm -w sysXyz-ui run ci
npm -w sysXyz-ui run preview
```
