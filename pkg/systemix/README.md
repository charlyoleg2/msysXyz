Systemix
========


Presentation
------------

*systemix* is the *core library* enabling the description and computation of the parameters of a system.


Links
-----

- [systemix](https://www.npmjs.com/package/systemix) : systemix as npm-package
- [systemcli](https://www.npmjs.com/package/systemcli) : systemcli as npm-package
- [systemui](https://www.npmjs.com/package/systemui) : systemui as npm-package
- [sources](https://github.com/charlyoleg2/msysXyz) : git-repository
- [github-pages](https://charlyoleg2.github.io/msysXyz/) : public instance of the UI


Getting started
---------------

```bash
git clone https://github.com/charlyoleg2/msysXyz
cd msysXyz
npm -w systemix install
npm -w systemix run ci
```
