sysXyz
======


Presentation
------------

*sysXyz* is a *system library* based on [systemix](https://www.npmjs.com/package/systemix).

It contains the parameters of the system *Xyz*.


Links
-----

- [sysXyz-ui](https://charlyoleg2.github.io/msysXyz/) : public instance of the UI
- [sources](https://github.com/charlyoleg2/msysXyz) : git-repository

- [pkg-uis](https://www.npmjs.com/package/sysXyz-uis) : sysXyz-uis as npm-package


Getting started
---------------

```bash
git clone https://github.com/charlyoleg2/msysXyz
cd msysXyz
npm -w sysXyz install
npm -w sysXyz run ci
```

Dependencies
------------

*sysXyz* runtime depends only on *Systemix*. Use *sysXyz-ui* WebUI or *sysXyz-cli* CLI to compute, check and export the system-parameters.
